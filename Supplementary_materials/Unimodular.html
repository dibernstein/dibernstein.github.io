Supplementary Materials for "Unimodular hierarchical models and their Graver bases"

This webpage contains supplementary material for the paper Unimodular hierarchical models and their Graver bases by Daniel Irving Bernstein and Christopher O'Neil. 

The file unimodularSupplement.m2 provides Macaulay 2 code that uses 4ti2 to verify the non-unimodularity of the last HM pair listed in Proposition 4.1. The code exhibits a Graver basis element of the corresponding matrix which has an entry of absolute value greater than 1.
